This project is a real-time hand tracking system built using OpenCV and MediaPipe. 
 
The system can recognize different hand gestures such as open hand, fist, thumbs up, peace sign, pointing. 

One interesting feature is the connection between two hands.

Technologies | Python | OpenCV | MediaPipe | NumPy

# Installation
   Clone the repository
 - git clone https://github.com/Nethika-Alagarathnam/HandGestures.git
 - cd HandGestures
 - pip install opencv-python mediapipe numpy
 - python hand_tracker.py
 - Press Q to quit application
   
Make sure your webcam is connected and working before running the program.
